# CLI Reminder
CLI Reminder is a simple command-line tool that allows users to set and manage reminders directly from the terminal. It supports scheduling reminders for specific dates and times, as well as recurring reminders.
## Features
- Set one-time reminbders

## How to Use
1. Install the CLI Reminder tool by following the installation instructions in the repository.
2. Open your terminal.  
3. Use the following command to set a reminder:
```bash
npm run cmd <feature> [options]
```

### Examples
- Set a one-time reminder:
```bash
npm run cmd remindme <reminder text> <time> <unit>
```
-view current tieme
```bash
npm run cmd time
```
